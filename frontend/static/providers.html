Gmail configuration

For Gmail: turn on 2-Step Verification, generate a 16-character App Password, and paste it below.

Gmail prerequisites
  1. Enable 2-Step Verification.
  2. Generate an App Password (16 characters).
  3. Paste it as App password. Your normal Gmail password will be rejected.
  4. Daily caps: ~500/day for personal Gmail, ~2,000/day for Workspace.
Custom SMTP

Use your provider's documented host/port. Common combos: 587 + STARTTLS or 465 + SSL.

Dry run
